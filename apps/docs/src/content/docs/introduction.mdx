## What is DocyUI?

This is **not a component library**. It's a collection of reusable components that you can copy and paste into your apps.

DocyUI is based on the excellent work of [shadcn/ui](https://ui.shadcn.com) and extends it with:

- **Multi-framework support** - Components for both React and Vue
- **Enhanced CLI tooling** - Streamlined component installation and management
- **11 color palettes** - Carefully crafted themes for every project
- **TypeScript first** - Full type safety and IntelliSense support

## Key Principles

### Copy and Paste

Components are designed to be copied and pasted directly into your project. No npm installs, no peer dependencies, no breaking changes.

- You own the code and can modify it as needed
- No external dependencies to manage
- Full control over styling and behavior

### Multi-Framework

Built for both React and Vue with consistent APIs and styling. Use the same components across different projects.

- React components with TypeScript support
- Vue 3 components with Composition API
- Consistent design language across frameworks

### Accessible

Built on top of Radix UI (React) and Radix Vue with full keyboard navigation, screen reader support, and ARIA attributes.

- Full keyboard navigation support
- Screen reader compatible
- WCAG 2.1 compliant

### Customizable

Built with Tailwind CSS and CSS variables. Easy to customize with your own design tokens and color schemes.

- CSS variables for easy theming
- 11 built-in color palettes
- Dark mode support out of the box
